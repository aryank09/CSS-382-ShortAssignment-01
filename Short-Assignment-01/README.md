## Idea

A **Fortune Cookie** web app built in a single HTML file using pure HTML, CSS, and JavaScript — no frameworks, no APIs.

The user clicks a CSS/SVG fortune cookie, which cracks open with a shake-and-split animation, then reveals a random philosophical-but-funny fortune. Fortunes are drawn from a shuffled deck of 47 strings so no fortune repeats until all have been seen. A copy-to-clipboard button and an "Another Fortune" button (which reassembles and re-cracks the cookie) round out the experience. The design uses a dark moody background, warm golden accent colors, twinkling stars, and elegant serif typography throughout.
